Css Grid 布局

2022/4/3 学习笔记 前端 CSS

简介

Grid 布局即网格布局，是一种新的 CSS 布局模型，比较擅长将一个页面划分为几个主要区域，以及定义这些区域的大小、位置、层次等关系。号称是最强大的的 CSS 布局方案，是目前唯一一种 CSS 二维布局。

Flex 和 Grid 的区别

Flex 为 一维布局 弹性布局
Grid 为 二维布局 网格布局

关于 Flex

Flex 的使用

    div{
        display: flex;
    }

关于 Grid

Grid 的使用

    div{
        display: grid;
    }

жңҖиҝ‘еңЁзңӢcss3жүӢеҶҢж—¶еҝҪз„¶еҸ‘зҺ°дәҶж–°зҡ„еҠЁз”»еұһжҖ§css, еҗҚеҸ«Motion PathпјҢеҲ©з”ЁMotion PathпјҢжҲ‘д»¬еҸҜд»ҘжҺ§еҲ¶е…ғзҙ жҢүз…§зү№е®ҡзҡ„и·Ҝеҫ„иҝӣиЎҢдҪҚзҪ®еҸҳжҚўзҡ„еҠЁз”»гҖӮе№¶дё”пјҢиҝҷдёӘи·Ҝеҫ„еҸҜд»ҘжҳҜйқһеёёеӨҚжқӮзҡ„дёҖжқЎи·Ҝеҫ„гҖӮ

CSS Motion Path и§„иҢғдё»иҰҒеҢ…еҗ«д»ҘдёӢеҮ дёӘеұһжҖ§пјҡ

  • offset-pathпјҡжҺҘж”¶дёҖдёӘ SVG и·Ҝеҫ„пјҲдёҺ SVG зҡ„pathгҖҒCSS дёӯзҡ„ clip-path зұ»дјјпјүпјҢжҢҮе®ҡиҝҗеҠЁзҡ„еҮ дҪ•и·Ҝеҫ„
  • offset-distanceпјҡжҺ§еҲ¶еҪ“еүҚе…ғзҙ еҹәдәҺ offset-path иҝҗеҠЁзҡ„и·қзҰ»
  • offset-positionпјҡжҢҮе®ҡ offset-path зҡ„еҲқе§ӢдҪҚзҪ®
  • offset-anchorпјҡе®ҡд№үжІҝ offset-path е®ҡдҪҚзҡ„е…ғзҙ зҡ„й”ҡзӮ№гҖӮ иҝҷдёӘд№ҹз®—еҘҪзҗҶи§ЈпјҢиҝҗеҠЁзҡ„е…ғзҙ еҸҜиғҪдёҚжҳҜдёҖдёӘзӮ№пјҢйӮЈд№Ҳе°ұйңҖиҰҒжҢҮе®ҡе…ғзҙ дёӯзҡ„е“ӘдёӘзӮ№йҷ„зқҖеңЁи·Ҝеҫ„дёҠиҝӣиЎҢиҝҗеҠЁ
  • offset-rotateпјҡе®ҡд№үжІҝ offset-path е®ҡдҪҚж—¶е…ғзҙ зҡ„ж–№еҗ‘пјҢиҜҙдәәиҜқе°ұжҳҜиҝҗеҠЁиҝҮзЁӢдёӯе…ғзҙ зҡ„и§’еәҰжңқеҗ‘

еҠЁжҖҒи§„еҲ’

еҠЁжҖҒи§„еҲ’дёҚжҳҜйӯ”жі•пјҢе®ғд№ҹжҳҜйҖҡиҝҮжҡҙеҠӣж–№жі•е°қиҜ•зӯ”жЎҲпјҢеҸӘжҳҜж–№ејҸжӣҙеҠ  вҖңиҒӘжҳҺвҖқпјҢдҪҝеҫ—е®һйҷ…дёҠж—¶й—ҙеӨҚжқӮеәҰе№¶дёҚй«ҳгҖӮ

еҠЁжҖҒи§„еҲ’дёҺжҡҙеҠӣгҖҒеӣһжәҜз®—жі•зҡ„еҢәеҲ«

дёҠйқўиҝҷеҸҘиҜқд№ҹиҜҙжҳҺдәҶпјҢжүҖжңүеҠЁжҖҒи§„еҲ’й—®йўҳйғҪиғҪйҖҡиҝҮжҡҙеҠӣж–№жі•и§ЈеҶіпјҒжҳҜзҡ„пјҢжүҖжңүжңҖдјҳи§Јй—®йўҳйғҪеҸҜд»ҘйҖҡиҝҮжҡҙеҠӣж–№жі•е°қиҜ•пјҲд»ҘеҸҠеӣһжәҜз®—жі•пјүпјҢжңҖз»ҲжүҫеҮәжңҖдјҳзҡ„йӮЈдёӘгҖӮ
жҡҙеҠӣз®—жі•еҮ д№ҺеҸҜд»Ҙи§ЈеҶідёҖеҲҮй—®йўҳгҖӮеӣһжәҜз®—жі•зҡ„зү№зӮ№жҳҜпјҢйҖҡиҝҮжҡҙеҠӣе°қиҜ•дёҚеҗҢеҲҶж”ҜпјҢжңҖз»ҲйҖүжӢ©з»“жһңжңҖдјҳзҡ„зәҝи·ҜгҖӮ
иҖҢеҠЁжҖҒи§„еҲ’д№ҹжңүеҲҶж”ҜжҰӮеҝөпјҢдҪҶдёҚз”ЁжҠҠжҜҸжқЎеҲҶж”Ҝе°қиҜ•еҲ°з»ҲзӮ№пјҢиҖҢжҳҜеңЁиө°еҲ°еҲҶеҸүи·ҜеҸЈж—¶пјҢеҸҜд»ҘзӣҙжҺҘж №жҚ®еүҚйқўеҗ„еҲҶж”Ҝзҡ„иЎЁзҺ°пјҢзӣҙжҺҘжҺЁеҜјеҮәдёӢдёҖжӯҘзҡ„жңҖдјҳи§ЈпјҒз„¶иҖҢж— и®әжҳҜзӣҙжҺҘжҺЁеҜјпјҢиҝҳжҳҜеүҚйқўеҗ„еҲҶж”ҜеҲӨж–ӯпјҢйғҪжҳҜжңүжқЎд»¶зҡ„гҖӮеҠЁжҖҒи§„еҲ’еҸҜи§Јй—®йўҳйңҖеҗҢж—¶ж»Ўи¶ід»ҘдёӢдёүдёӘзү№зӮ№пјҡ
